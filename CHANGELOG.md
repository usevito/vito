# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-22

### Added
- MCP proxy server with stdio transport
- Rules engine with tool blocking, budget limits, and approval gates
- Prompt injection detection
- Circuit breaker for rate limiting
- Semantic analysis for high-risk operations
- Persistent audit logging
- Web dashboard with real-time status
- Emergency kill switch
- Test MCP server for development
- Automated end-to-end test suite (16 tests)
- Interactive demo mode
- Docker support
